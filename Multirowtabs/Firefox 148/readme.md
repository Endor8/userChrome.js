### MultiRowTabs@BuF.uc.js ###

The script has been completely rebuilt;      
the CSS code has been entirely removed and moved to a separate CSS file       
(MultiRowTabs@BuF.css). While the script can technically run without this file,      
the tabs will lack their full appearance, and some functions within the     
script itself will be ineffective. Therefore, it's advisable to use both      
files together, allowing you to customize the CSS code to your liking.
